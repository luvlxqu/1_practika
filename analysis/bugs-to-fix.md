# Список багов и улучшений для Дня 5

## 🔴 Критические ошибки (E — Error)

### E0611: Проблемы с импортами (36 шт.)
**Описание:** No name 'X' in module 'Y' — импортируемый объект не найден.
**Причина:** Возможно, классы или функции были переименованы, удалены или pylint не видит динамические импорты.

## 🟠 Важные предупреждения (W — Warning)

### W1203: Неправильное логирование (24 шт.)
**Описание:** Use lazy % formatting in logging functions.

## 🟡 Стилистические замечания (C — Convention)

### C0301: Длинные строки (27 шт.)
**Описание:** Line too long (>100 символов).
**План:** Разбить длинные строки на несколько.

### C0114: Отсутствует docstring модуля (22 шт.)
**Описание:** Missing module docstring.
**План:** Добавить docstring в начало каждого файла.

## 🔵 Рекомендации по рефакторингу (R — Refactor)

### R0903: Слишком мало публичных методов (19 шт.)
**Описание:** Too few public methods.
**Решение:** Игнорировать (особенность Flask-приложений).
